# Gota a Gota Control - Guía de Extensión Nativa

Esta carpeta contiene el código fuente de la **Extensión Nativa** diseñada para el monitoreo persistente 24/7.

## Contenido:
- `manifest.json`: Configuración de permisos de Chrome.
- `background.js`: Servicio que mantiene la conexión viva incluso si la pestaña se cierra.

## Cómo Instalar:
1. Descargue el archivo ZIP desde la aplicación (Botón "INSTALAR EXTENSIÓN").
2. Descomprima el archivo.
3. En Chrome, vaya a `chrome://extensions/`.
4. Active el **Modo de desarrollador**.
5. Haga clic en **Cargar descomprimida** y seleccione la carpeta.

Esta extensión permite que la aplicación web se comporte como una aplicación nativa de control parental, garantizando que el GPS y la pantalla se transmitan sin interrupciones del sistema operativo.
